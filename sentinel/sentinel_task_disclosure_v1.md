# Sentinel Task Disclosure v1 (Template)

Architect: ONLY the Architect can approve.
Execution: BLOCKED until explicit approval.

FIELDS (ALL REQUIRED):
- TASK ID
- TITLE
- OBJECTIVE
- SCOPE (allowed / forbidden)
- PRE-FLIGHT CHECKS
- PROPOSED TASK LIST (one action per line)
- COMMANDS (PLAN ONLY — DO NOT RUN)
- EXPECTED OUTPUTS
- VALIDATION / PROOF
- ROLLBACK
- STOP CONDITIONS
- RISK LEVEL

APPROVAL PHRASE (exact):
"APPROVED – EXECUTE TASK ID <ID>"
