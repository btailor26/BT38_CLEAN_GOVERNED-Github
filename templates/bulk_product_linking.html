Column Definitions
sku
Your internal warehouse SKU (must exist in warehouse_stock)
asin
Amazon Standard Identification Number (10 characters)
fulfillment_type
MFN (Merchant Fulfilled) or FBA (Fulfilled by Amazon)
FAQ / Help
What does this page do?
  • Links warehouse SKUs to Amazon ASINs in bulk via CSV upload
  • Creates or updates marketplace_listings records for each SKU
  • Enables stock sync from warehouse to Amazon for linked products
Main Workflow
  1. Select your Amazon store from the dropdown
  2. Prepare a CSV file with sku, asin columns
  3. Upload the CSV file
  4. Review the results (success/error counts)
  5. Download error log if any rows failed
Common Errors
  • SKU not found: The SKU doesn't exist in warehouse_stock. Add it first.
  • Invalid ASIN format: ASIN must be exactly 10 characters.
  • Missing columns: CSV must have at least sku and asin columns.
Where are Bulk Actions?
  • This IS the bulk action page for product linking
  • After linking, use Warehouse → Push Selected to sync quantities
