Upload Amazon Export
  1. Go to Amazon Seller Central → Inventory → Manage All Inventory
  2. Select all listings (or use filters)
  3. Copy the entire page content (Ctrl+A, Ctrl+C)
  4. Paste into a text file and save as .txt
  5. Upload the file here
Accepts .txt or .csv files from Amazon Seller Central
About This Feature
Why Manual Import?

When Amazon API authorization is unavailable (OAuth issues, pending approvals), you can manually sync inventory using Seller Central exports.

How It Works
  • Parse your Amazon export file to extract SKUs and quantities
  • Compare with current warehouse quantities
  • Review mismatches before applying changes
  • Choose to apply all or selected updates
Data Extracted
  • SKU and ASIN
  • Product name
  • Available quantity (FBA/FBM)
  • Price and listing status
Note: This updates the warehouse stock to match Amazon. Review carefully before applying changes.
