Control Center FAQ

What is Global Push? The master switch checked by the runtime gate before push/sync/import work is allowed.
What is a Webhook? A marketplace notification that can reduce polling by telling BT38 something changed.
What is Polling Fallback? Scheduled checking used when webhooks are off, unavailable, or not trusted yet.
What is Runtime Gate? The central backend safety check used by routes, queues, dispatcher, and services.
Can any admin use this? Yes. Each row shows the action, backend field/API, current state, control, and plain-English meaning.
What should not be changed casually? Marketplace active state, FBM push, webhook worker, and emergency controls should only be changed when the operator understands the effect.
