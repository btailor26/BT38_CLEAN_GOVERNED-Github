CSV Format Requirements

Your CSV file must have headers in the first row. Required and optional columns:

Column Name Required Description
single_sku Yes SKU of the individual product
units_per_carton Recommended Number of units in each carton (default: 1)
single_barcode Optional Barcode of individual product
master_barcode Optional Barcode on the master carton
carton_weight_kg Optional Weight of full carton in kg
carton_length_cm Optional Carton length in cm
carton_width_cm Optional Carton width in cm
carton_height_cm Optional Carton height in cm
notes Optional Any notes about the carton
Example CSV:
single_sku,units_per_carton,master_barcode,carton_weight_kg
SKU-001,12,CTN-001-12,5.5
SKU-002,24,CTN-002-24,8.2
SKU-003,6,CTN-003-06,2.1
Note: If a mapping for a SKU already exists, it will be updated with the new values from the CSV.
